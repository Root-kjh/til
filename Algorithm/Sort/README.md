# 정렬(Sort)
> n 개의 숫자가 입력으로 주어졌을 때, 이를 사용자가 지정한 기준에 맞게 정렬하여 출력하는 알고리즘
    

## 정렬의 종류

1. 선택 정렬(Selection Sort)

2. 삽입 정렬(Insertion Sort)

3. 버블 정렬(Bubble Sort)

4. 병합 정렬(Merge Sort)

5. [퀵 정렬(Quick Sort)](/Algorithm/Sort/Quick_Sort)

6. 힙 정렬(Heap Sort)

7. 팀 정렬(Tim Sort)

8. 인트로 정렬(Intro Sort)

9. 기수 정렬(Radix Sort)

10. 카운팅 정렬(Counting Sort)
