# Django
---

    파이썬으로 이루어진 무료 오픈소스 웹 프레임워크

## Django의 동작 구조

* Request 요청이 Django로 전달.

* urlresolver는 url을 가져와 일치하는 패턴의 함수(view)에 전달.

* 권한 확인 후 Response return

## Django 설치

```
pip install Django
```

## Django 환경구성

```
django-admin startproject project-name

python manage.py startapp app-name
```
